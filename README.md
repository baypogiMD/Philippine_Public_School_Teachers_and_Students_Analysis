# Philippine Public School Teachers and Students Analysis

## Overview
This repository provides a comprehensive analytical framework for examining
public school teacher and student data in the Philippines. The project supports
education research, policy evaluation, and resource planning using official
open data published via Kaggle and data.gov.ph.

## Dataset Source
- Kaggle: https://www.kaggle.com/datasets/franksebastiancayaco/philippine-public-school-teachers-and-students

## Objectives
- Quantify trends in enrollment and teacher deployment
- Evaluate teacher–student ratios across regions and school categories
- Identify regional inequalities in education resources
- Model future staffing and enrollment needs
- Translate analytics into policy-relevant insights

## Analytical Coverage
1. Descriptive & Summary Statistics
2. Time Series Trend Analysis
3. Teacher–Student Ratio Analysis
4. Geographic & Regional Comparisons
5. Category-Level Comparisons
6. Correlation & Relationship Analysis
7. Regional Inequality Metrics
8. Change Point & Policy Impact Analysis
9. Advanced Statistical Modeling
10. Visualization & Reporting
11. Policy Implications & Contextual Integration

## Tools & Libraries
- Python 3.9+
- pandas, numpy
- matplotlib, seaborn, plotly
- statsmodels, scipy
- scikit-learn
- geopandas (optional for maps)
